No-server Admin Panel

Update content, preview locally, then export for GoDaddy

This editor runs fully in the browser. Save keeps a private preview on this computer. Export downloads `site-data.json`; upload that file to `assets/data/site-data.json` on GoDaddy to publish text changes.

Programmes

Gallery

For a fully static site, uploaded images are stored in your browser preview. To publish new photos, also upload the image files into `assets/images` and use that path here.

Publishing Steps

  1. Make text and programme changes here.
  2. Use Save Preview to check this browser.
  3. Use Export JSON.
  4. Upload the downloaded file to `assets/data/site-data.json` in GoDaddy File Manager.
  5. For new photos, upload the image file to `assets/images` and keep the matching path in Gallery.
